Displays a widget with bilingual post Calendar. The Bangla Date plugin (Version 1.2) is required with this. (http://wordpress.org/extend/plugins/bangla-date/)

== Description ==
Gives a Calendar with both Bongabdo and Gregorian Calendar.

== Installation ==

1. Install Bangla Date first (`http://wordpress.org/extend/plugins/bangla-date/`)
1. Upload the plugin in your wordpress plugin directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the widget selection page of your theme
